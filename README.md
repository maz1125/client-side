## front-side-page-snippet について  

### 1.概要  
front-side-page-snippetは下記のsnippetを提供します。  
1.pages/_page_snippet(ポータルページ、入力ページの大枠)    
2.pages/_snippet(画面上に配置することのできるParts)  

利用する際は、_page_snippetを利用して画面の大枠を作成した後、必要なPartsを_snippetから選んで画面に配置してください。  
利用方法は、各フォルダ直下においてあるREADMEを参照してください。

### 2.構成  
HTML,json,JavaScript,CSSで作られています。
